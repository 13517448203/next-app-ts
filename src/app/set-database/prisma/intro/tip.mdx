# Prisma
[Prisma](https://www.prisma.io/docs/getting-started/quickstart) 是一个现代化的数据库工具，它提供了类型安全的数据库访问和自动生成的查询客户端。Prisma 具有 ORM（对象关系映射）的特性，但它不是传统意义上的 ORM。
