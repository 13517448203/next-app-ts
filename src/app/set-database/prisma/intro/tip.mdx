# [Prisma](https://www.prisma.io/docs/getting-started/quickstart)
Prisma 是一个现代化的数据库工具，它提供了类型安全的数据库访问和自动生成的查询客户端。Prisma 具有 ORM（对象关系映射）的特性，但它不是传统意义上的 ORM。

# [ORM](https://www.prisma.io/docs/orm)
ORM（对象关系映射）是一种编程技术，用于在关系型数据库和面向对象编程语言之间建立映射关系。ORM 可以将数据库表映射为编程语言中的对象，简化了数据库操作和数据访问的过程。

# 区别
- Prisma 的主要目标是提供类型安全、自动生成代码和现代化的数据库访问方式，以及强大的数据建模和查询能力。
- ORM 是一种编程技术，用于在关系型数据库和面向对象编程语言之间建立映射关系。

# 优势
- 类型安全：使用 Prisma 编写的代码可以获得类型检查和自动补全，确保代码的正确性和可维护性。
- 自动生成代码：Prisma 可以根据数据库模式自动生成查询客户端，无需手动编写代码。
- 现代化的数据库访问方式：Prisma 提供了一种简单、高效的方式来访问数据库，无需手动编写 SQL 语句。

# 劣势
- 学习曲线陡峭：Prisma 是一个新的工具，需要学习新的概念和语法，有一定的学习成本。
- 性能问题：Prisma 可能存在性能问题，需要进行性能测试和优化。
