# Next.js 介绍
- 当前项目用的是 Next.js 的 App 路由，[App Router](https://nextjs.org/docs/getting-started/installation)。

# [链接与导航](https://nextjs.org/docs/app/building-your-application/routing/linking-and-navigating)
## 路由导航方式
- 页面级路由：使用 Link 组件。
- 应用程序级路由：使用 Router 对象。

# 其他
## [MDX](https://nextjs.org/docs/app/building-your-application/configuring/mdx)
- MDX 是一种结合了 Markdown 和 JSX 的格式，Next.js 允许在 Markdown 文件中编写 JSX 组件。
