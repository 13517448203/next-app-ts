# 链接和导航
Next.js 有四种在路由之间导航的方法：
- 使用 `<Link>` 组件
- 使用 `useRouter` 钩子（客户端组件）
- 使用 `redirect` 功能（服务器组件）
- 使用 `window.history`

## [Link](https://nextjs.org/docs/app/building-your-application/routing/linking-and-navigating#link-component)
`<Link>` 是一个内置组件，页面级路由之间的导航方式，在生产环境中，每当 `<Link>` 组件出现在浏览器的视口中时，Next.js 就会在后台自动预取链接路由的代码

## [useRouter](https://nextjs.org/docs/app/building-your-application/routing/linking-and-navigating#userouter-hook)
`useRouter` 允许以编程方式更改客户端组件的路由。

## [redirect](https://nextjs.org/docs/app/building-your-application/routing/linking-and-navigating#redirect-function)
`redirect` 允许服务器组件和 API 路由进行客户端重定向。
